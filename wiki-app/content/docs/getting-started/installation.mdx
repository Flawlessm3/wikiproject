## System Requirements

| Requirement | Minimum | Recommended |
|-------------|---------|-------------|
| Node.js     | 18.x    | 20.x LTS    |
| npm         | 8.x     | 10.x        |
| RAM         | 512 MB  | 1 GB+       |
| Disk space  | 300 MB  | 500 MB      |

## Installing Node.js

### Option A — Official installer

Download the LTS installer from [nodejs.org](https://nodejs.org) and follow the setup wizard.

### Option B — Using nvm (recommended)

`nvm` lets you switch between Node.js versions easily:

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# Install and use Node 20 LTS
nvm install 20
nvm use 20
```

### Verify installation

```bash
node --version   # v20.x.x
npm --version    # 10.x.x
```

## Installing the Project

```bash
# Navigate to the project directory
cd wiki-app

# Install all dependencies
npm install
```

This installs:
- **Next.js 14** — React framework with App Router
- **Tailwind CSS** — Utility-first styling
- **next-mdx-remote** — MDX rendering in RSC
- **gray-matter** — Frontmatter parsing
- **next-themes** — Dark/light theme management
- **lucide-react** — Icon library

## Environment Variables

This project requires no environment variables out of the box. If you add a backend or analytics later, create `.env.local`:

```bash
# .env.local (not committed to git)
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

<Callout type="note">
  `.env.local` is already in `.gitignore` by Next.js convention. Never commit secrets to version control.
</Callout>

## Troubleshooting

### Port already in use

```bash
# Use a different port
npm run dev -- --port 3001
```

### Module not found errors

Delete `node_modules` and reinstall:

```bash
rm -rf node_modules .next
npm install
```

### TypeScript errors on first run

Run the type checker:

```bash
npx tsc --noEmit
```

<Callout type="warning">
  If you see `Cannot find module '@/...'` errors, ensure your `tsconfig.json` has the correct `paths` configuration pointing `@/*` to `./src/*`.
</Callout>
